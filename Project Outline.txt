Log Formatter

Primary Purpose:
To display specified logs from the log output when running Valence in a clear, concise manner, being able to change output based on the user's need.

Requirements:
1. Quickly display logs as they are generated by Valence suite
2. Allow user to specify what logs to view, i.e. INFO, DEBUG, STEPS, etc.
3. Colorize specific portions of the log for clear visibility
4. Application to be easily accessible

Hurdles
1. Retrieving and displaying logs in web application
2. Automatically minimizing portions of logs that contain info not readily needed.
3. Interactively maximizing portions of logs that have been minimized.
4. Have settings for user-specified logs
